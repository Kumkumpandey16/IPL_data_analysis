# 🏏 IPL Data Analysis Dashboard

## 📌 Project Overview

This project analyzes IPL cricket match data using Python, SQL Server, and Power BI. The objective is to clean the dataset, perform exploratory data analysis, write SQL queries for insights, and build an interactive dashboard.

---

## 🛠 Tools & Technologies

- Python
- Pandas
- NumPy
- SQL Server
- Power BI
- Git
- GitHub
